# Tournament Manager 🏆

A web-based tournament management system for 6 teams with group stage and knockout rounds.

## Features
- ✅ Round-robin group stage with 6 teams
- ✅ Live table updates as you enter scores
- ✅ Head-to-head tiebreaker for tied points
- ✅ Semi-finals and final generation
- ✅ Reset button to clear all scores
- ✅ Responsive design with clean styling

## How to Use
1. Enter scores for each match in the group stage
2. The league table updates automatically
3. Semi-finals and final are generated based on final standings
4. Use the reset button to start over
